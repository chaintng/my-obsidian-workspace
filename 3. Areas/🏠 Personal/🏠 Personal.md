## Last updated files
```dataview
TABLE WITHOUT ID
file.link AS Files, file.mtime as "Last Modified"
FROM "3. Areas/🏠 Personal"
WHERE file.name != this.file.name
  AND !contains(file.path, "ADHOC")
  AND !contains(file.tags, "hidden")
SORT file.mtime desc
LIMIT 20
```
